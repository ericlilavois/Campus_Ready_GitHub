**Purpose:** Living reference document for all active, pending, and prospective corporate and organizational partners. Read this before drafting any communication that involves a partner relationship.

## How to Use This Document

Each partner entry covers: what they provide, key contacts, approved public language, protocol notes (rules that must be followed), outstanding items, and the current next move. Update this document whenever a partner relationship materially changes — new contacts, new commitments, new constraints.

**Critical rules that apply across all partners:**
- Never share grant or gift dollar amounts across partner relationships. Each partner's contribution is confidential relative to other partners.
- Never claim "partnership" before it's earned. Match language to the actual state of the relationship.
- Never add a partner's logo to the site or post publicly about them without explicit approval.
